# Smart City Platform — Uralsk

Цифровая экосистема "Умного города" для автоматизации городских процессов, сквозной аналитики и оперативного взаимодействия между гражданами, коммунальными службами и ситуационным центром г. Уральск.

---

## 1. Актуальность и проблематика
Проект направлен на решение ключевых инфраструктурных проблем урбанистики:
*   **Разрозненность данных:** Отсутствие единой панели мониторинга для городских служб.
*   **Низкая скорость реагирования:** Долгий цикл обработки заявок от населения по коммунальным и дорожным инцидентам.
*   **Отсутствие прозрачности:** Граждане не видят реальный статус выполнения работ по городским проблемам.

## 2. Функционал платформы
*   **Ситуационный дашборд:** Интерактивные графики и аналитические чарты для мониторинга состояния инфраструктуры в реальном времени.
*   **Модуль управления инцидентами:** Полноценная CRUD-система для регистрации, обработки и трекинга городских заявок.
*   **Система авторизации:** Разграничение прав доступа (Гражданин / Сотрудник служб / Администратор города) через защищенный протокол.
*   **Интеграция с СУБД:** Хранение истории инцидентов, логов и геоданных с поддержкой Realtime-обновлений.

## 3. Технологический стек (Architecture & Tech Stack)
Платформа построена на базе современного, производительного и масштабируемого стека:
*   **Frontend & SSR Handler:** TanStack Start, Vite (сборщик), Tailwind CSS v4 (стилизация интерфейса), TypeScript (строгая типизация).
*   **Backend & Database:** Supabase (PostgreSQL в качестве реляционной СУБД, Row Level Security для защиты данных, встроенный сервис Auth для управления сессиями).
*   **Runtime Environment:** Разработка оптимизирована под среду выполнения **Bun** для максимальной скорости сборки и работы серверных функций.
*   **Хостинг / Деплой:** Серверная и клиентская части развернуты в бессерверной среде **Vercel** с оптимизацией SSR-обработчиков.
